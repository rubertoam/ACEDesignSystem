# ACE Design System

React component library and design tokens for the ACE / FinScan design system.

## Install

```bash
npm install @rubertoam/ace-design-system
```

## Usage

Import styles once in your app entry:

```ts
import '@rubertoam/ace-design-system/style.css'
```

Import components:

```tsx
import { AceButton, AceTimeline } from '@rubertoam/ace-design-system'
```

## Lab site

Interactive documentation: https://rubertoam.github.io/ACEDesignSystem/

- `react` (18 or 19)
- `react-dom` (18 or 19)
